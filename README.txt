Git VCS Extension for Komodo



Homepage and source available: http://github.com/nevergone/gitext

Supported Komodo Edit and Komodo IDE versions: 4.1 - 5.x


Howto build:

Windows (command-line): 
"<installed Komodo>\lib\python\python.exe" "<installed Komodo>\lib\sdk\bin\koext.py" build --unjarred

Linux:
<installed Komodo>\lib\sdk\bin\koext.py build --unjarred

After successful completion an .xpi file is created at the same location. Add this to Komodo, just like any other extension.
