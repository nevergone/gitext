Git VCS kiegészítő Komodohoz



A projekt honlapja és forrása: http://github.com/nevergone/gitext

Támogatott Komodo Edit és Komodo IDE verziók: 4.1 - 5.x


Segítség lefordításhoz:

Windows alatt parancssorból: 
"<telepített Komodo>\lib\python\python.exe" "<telepített Komodo>\lib\sdk\bin\koext.py" build --unjarred

Linux alatt:
<telepített Komodo>\lib\sdk\bin\koext.py build --unjarred

Sikeres művelet esetén létrejött ugyanott egy .xpi kiterjesztésű fájl, ezt add hozzá a Komodohoz, mint a többi kiterjesztést.
